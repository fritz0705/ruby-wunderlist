h1. ruby-wunderlist

ruby-wunderlist is an inofficial Ruby binding for the undocumented Wunderlist API.

Features:
  - Change list names
  - Get all tasks of a list

h2. Dependencies

* "json":http://rubygems.org/gems/json
* "nokogiri":http://rubygems.org/gems/nokogiri

h2. Example

<pre>
  require "wunderlist"
  api = Wunderlist::API.new
  
  # Login to Wunderlist (Returns true if successful)
  api.login "johndoe@example.org", "mypassword"
  
  # Get all lists of John Doe
  lists = api.lists
  
  # Get the first list (This should be Inbox)
  list = lists.to_a.first[1]

  # Get all tasks of the first list
  tasks = list.tasks
  
  # Rename list
  list.name = "Something like Inbox"

  # Save list
  list.save
</pre>
